# Athletics Pace Calculator

A simple static web app for calculating running pace, average speed, and split times.

## Features

- Enter distance in meters
- Enter total time in minutes and seconds
- Calculate pace per kilometer
- Calculate average speed in km/h
- Calculate split times for 200m, 400m, 600m, 800m, and 1000m

## Open Locally

1. Open the project folder.
2. Double-click `index.html`.
3. The app will open in your default web browser.

No install step is required. The app uses only HTML, CSS, and JavaScript.

Deployed with GitHub Pages as a PWA.
